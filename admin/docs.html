产品简介

PDF 简历智能管理系统是一套完整的简历收集、解析和管理解决方案，由Chrome 浏览器插件和Web 管理后台两部分组成。

在招聘网站上浏览简历时，通过插件一键上传 PDF 到云端服务器，系统会调用 AI（智谱 GLM）自动解析简历内容，提取出姓名、手机、邮箱、应聘职位、学历、经验、工作经历等结构化字段。你可以在管理后台统一查看、搜索、编辑和下载所有简历。

插件安装

1. 下载插件

点击顶部「下载插件」按钮，或在管理后台右上角点击「下载插件」，获取 chrome-resume-plugin.zip。

2. 解压

将下载的 zip 文件解压到本地任意目录，解压后得到一个 dist 文件夹。

3. 安装到 Chrome

  1. 1
    打开扩展管理页面
    在 Chrome 地址栏输入 chrome://extensions/ 并回车。
  2. 2
    开启开发者模式
    页面右上角打开「开发者模式」开关。
  3. 3
    加载扩展
    点击「加载已解压的扩展程序」，选择解压后的 dist 文件夹。
  4. 4
    固定到工具栏
    点击 Chrome 右上角拼图图标，找到「简历收集(本地)」，点击图钉固定到工具栏。

插件使用

登录账户

点击浏览器工具栏中的插件图标，弹出登录面板。首次使用需要注册账号（点击底部「没有账号？立即注册」），输入用户名和密码完成注册后即可登录。

注意：如果使用线上服务器，请将顶部「服务器」下拉框切换为「生产 (43.167.211.80)」，然后再登录/注册。本地测试使用「本地 (localhost:3000)」。

上传简历

  1. 1
    打开插件面板
    点击浏览器工具栏中的插件图标，确认已登录状态。
  2. 2
    选择 PDF 文件
    点击「选择文件」按钮，或在 Mac 上使用快捷键 ⌘+Shift+U（Windows: Ctrl+Shift+U）触发快速上传。
  3. 3
    等待上传完成
    上传进度条完成后，系统会自动调用 AI 解析简历信息。你可以打开管理后台查看结果。

快捷键

插件支持快捷键上传：Mac 使用 ⌘+Shift+U，Windows 使用 Ctrl+Shift+U。在任意页面按下快捷键即可触发文件选择器。

管理后台

管理后台通过浏览器访问：http://localhost:3000/admin（本地）或 http://43.167.211.80:3000/admin（线上）。

登录后台

使用与插件相同的用户名和密码登录。顶部可以切换连接的服务器（本地 / 生产），切换后会自动跳转到对应环境。

文件列表

登录后进入文件管理列表，可以看到所有已上传的 PDF 简历。表格展示以下信息：

序号、姓名、手机、邮箱、应聘职位、学历、工作经验、工作经历、上传时间、操作

其中工作经历列默认显示前两行，点击「展开」可查看全部。

搜索

通过顶部的搜索框，可以按文件名、姓名、手机、邮箱、应聘职位、学历等多字段模糊搜索。

分页

表格底部支持分页功能，可选择每页显示 10 / 20 / 50 条记录，并配有页码导航。

在线编辑

提示：鼠标悬停在手机、邮箱、应聘职位、学历、经验列上，单元格会出现蓝色虚线边框，点击即可直接编辑，修改后自动保存。

简历解析

上传的 PDF 文件会自动触发 AI 解析。解析中的文件会显示加载动画，解析完成后自动显示姓名、手机等解析结果。如果解析失败，可以手动点击「解析」按钮重试。

预览与下载

每条记录的操作列提供以下功能：

  1. 1
    预览
    点击「预览」弹出模态框，在浏览器中直接查看 PDF 内容，支持翻页浏览。
  2. 2
    下载
    点击「下载」保存 PDF 到本地，文件名自动生成为「姓名_电话_岗位.pdf」的格式，方便归档。
  3. 3
    删除
    点击「删除」移除该条记录，会同时删除云端存储的 PDF 文件（不可恢复）。

上传文件

在管理后台也可以直接上传 PDF 文件：点击页面顶部上传区域的「点击上传或拖拽 PDF 到此处」即可选择文件上传，也支持拖拽文件到该区域。上传后自动解析。

常见问题

1. 安装插件后点击图标没有反应？

请在 Chrome 扩展管理页面确认「简历收集(本地)」已启用。如果已经启用，尝试点击右上角刷新按钮，或移除后重新加载。

2. 登录失败怎么办？

首先确认选择了正确的服务器。如果是线上服务器，请确保能访问 http://43.167.211.80:3000。如果忘记密码，目前需要联系管理员重置。

3. 上传的文件去哪里了？

PDF 文件存储在腾讯云 COS（对象存储）中，数据库记录关联了文件的存储地址。你可以通过管理后台预览和下载。

4. 简历解析失败怎么办？

在管理后台找到对应记录，点击「解析」按钮手动重试。如果多次失败，可能是 PDF 文件过于复杂或包含图片，无法提取文本。

5. 解析结果不准确？

可以在管理后台直接点击对应字段进行编辑修改。鼠标悬停在手机、邮箱、职位、学历等列上，点击即可编辑。

6. 支持哪些 PDF 格式？

支持标准的文本型 PDF（含中英文）。纯图片扫描版 PDF 需要 OCR 支持，目前暂不支持。

7. 如何切换本地和服务器？

插件登录页和管理后台登录页顶部都有「服务器」下拉框，选择后会自动切换。注意切换服务器后需要重新登录。
